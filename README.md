The 'project code' folder contains the essential code needed to run the river herring visualization in any Leaflet map.

'data.csv' contains the PIT tag detections.

NOTE: the reference to the 'routes.js' file in index.php needs to be updated when this code is used

River Herring Visualization Web Map: https://gisviz.mit.edu/maps/river_herring/

River Herring Visualization Web Map description on MIT Sea Grant website: https://seagrant.mit.edu/gisviz/#river-herring
